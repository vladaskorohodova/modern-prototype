# Installation

Get started with the Modern React Knowledge System by installing the necessary packages and dependencies.

## Prerequisites

Before you begin, make sure you have the following installed:

- **Node.js** 18.17 or later
- **npm**, **yarn**, or **pnpm** package manager

## Installation

Install the package using your preferred package manager:

```bash
npm install modern-react-knowledge
```

Or with yarn:

```bash
yarn add modern-react-knowledge
```

Or with pnpm:

```bash
pnpm add modern-react-knowledge
```

## Quick verification

After installation, you can verify everything is working by importing the package:

```tsx
import { Button, Grid, Scheduler } from 'modern-react-knowledge';

export default function App() {
  return (
    <div>
      <Button variant="solid">Click me</Button>
    </div>
  );
}
```
