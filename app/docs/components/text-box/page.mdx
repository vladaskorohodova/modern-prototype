# TextBox Overview

Use the TextBox component to collect short, single-line user input (names, emails, search queries, IDs). It supports common states like disabled and error, and should always be paired with an accessible label.

```tsx
import { TextBox } from 'modern-react-knowledge';
```

## Accessibility

- Always add an associated label (visible or via `aria-label`).
- Use `aria-invalid` and an error message when validation fails.
- Don’t rely on placeholder text as the only label.

## Usage guidelines

### When to use

- Single-line input: names, emails, usernames
- Search inputs

### Best practices

- Validate on blur or submit (avoid overly aggressive validation while typing)
- Use helpful error messages and examples
- Use appropriate input type (e.g., `email`, `password`)
