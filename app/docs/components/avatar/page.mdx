# Avatar Overview

Use the Avatar component to represent a person or entity. Avatars typically display a profile image, but must gracefully fall back to initials when the image is missing.

```tsx
import { Avatar } from 'modern-react-knowledge';
```

## Accessibility

- Use meaningful `alt` text when the image conveys identity.
- If using a purely decorative avatar, set `alt=""` and ensure the nearby name adds context.
- Ensure sufficient contrast for fallback initials.

## Usage guidelines

### When to use

- User menus, profile headers, comments, and participant lists

### Best practices

- Keep fallback initials to 1–2 characters
- Use consistent sizing within a list
- Avoid relying on color alone to differentiate users
