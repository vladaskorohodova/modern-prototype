# CheckBox Overview

Use the CheckBox component for independent boolean selections (on/off). Checkboxes are typically used for settings, opt-ins, and confirming an action.

```tsx
import { CheckBox } from 'modern-react-knowledge';
```

## Accessibility

- Use a single clickable label for the checkbox and its text.
- Use clear, specific label text (avoid vague labels like “Enabled”).
- For validation, connect error text via `aria-describedby`.

## Usage guidelines

### When to use

- Multiple independent selections
- Consent/confirmation (e.g., “I agree…”) when appropriate

### Best practices

- Prefer radio buttons for mutually exclusive choices
- Keep labels short and state-based (what will happen when checked)
